# 懒人交易系统 (Lazy-Trader System)

这是一套我自用的期货交易操作面板,经过近十年的反复打磨,目前用得还挺顺手的.<br>

独乐乐不如众乐乐,想着应该会有人也需要的,于是断断续续花了大半年时间把它大力重构了几次,看起来没啥大毛病了,喜欢的朋友可以直接拿去用.<br>

我的交易:<br>
之前主要做港股小恒指期货,只做日内,算是波段或短线交易者,用的均线指标,再参考水平压力线,纯手工打单.<br>
近年因为不好明说的原因,想着迟早要把资金整回来的,就深入学习了一下大陆期货交易,这边与港期有很大的不同,目前还在适应阶段.<br>
短线动用资金20万内,交易水平很一般,只能说应该不至于大亏了,这点还是有信心的,客观地说还只能算是小学生.<br>

项目特点:<br>
a,接入富途OpenAPI[港股]和天勤量化[大陆]的实时行情<br>
b,实现个性化的K线图,固定刻度尺<br>
c,用按钮点击实现下单撤单,拖动划线实现改单<br>
d,浏览器端实现模拟交易,最大限度贴近实战<br>

运行步骤:<br>
1,Windows系统上,把文件下载过去,双击 lazy-trader.exe 运行,此时应该有启动成功的提示信息<br>
2,Chrome浏览器打开链接: http://127.0.0.1:10086 应该能看到一个期货列表的页面<br>
3,点击任一大陆期货,应该就能看到图表进行模拟交易了<br>
4,如果要进行天勤实盘或富途交易,则要修改 static/js/config.js<br>

一套专为追求简约而设计的交易辅助方案。系统采用 **JavaScript** 驱动图形化策略计算与交易执行，并由 **Go** 构建轻量化静态 Web 服务器，确保毫秒级的分发响应。

## 🚀 技术特性
- **计算核心**: 基于 JavaScript 的高性能图形化计算引擎，实现直观的策略反馈。
- **可视化**: 集成 **AnyChart** 商业级图表库，提供毫秒级 K 线渲染与技术指标展示。
- **分发后端**: Go 编写的 Web Server，零依赖部署，专为本地环境优化。

## 💻 运行环境
- **操作系统**: Windows 10 及以上版本（64位）。
- **推荐浏览器**: **Google Chrome** (为了获得最佳的渲染性能与兼容性)。
- **网络条件**: 建议开启稳定低延迟的网络代理以确保交易指令及时送达。
